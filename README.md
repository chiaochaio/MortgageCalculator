**詳細介紹如下:**<br><br>
左側輸入貸款詳情為使用者輸入基本資料的地方:<br>
輸入可使用Tab依序往下填寫，Shift+Tab可往回<br>

2.填入比例或金額會自動算出另一項的數值<br>
3.4.均可選擇使用快捷鍵或直接點選button來操作<br><br>
右側計算結果為輸入資料Enter後的結果顯示區:<br>
均有格式化，含千分位逗號與小數點後兩位<br>
<img width="490" height="480" alt="image" src="https://github.com/user-attachments/assets/79673bae-4189-4b8a-b2b2-d25ee73bfe03" /><br>

2.顯示所有計算出來的結果<br>
3.視覺化圓餅圖，讓使用者更清楚本金利息的占比
